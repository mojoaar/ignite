What is Ignite?

Ignite is a desktop GUI application for macOS that provisions new software projects using AI-guided conversational interviews. It produces four professional documentation files — a full project specification, an agent guide for AI assistants, an implementation plan, and a polished public README.

Why Ignite?

  • Save hours of boilerplate documentation writing
  • Consistent quality across all your projects
  • Bring your own AI provider — no vendor lock-in
  • Persistent history — resume projects anytime
  • Privacy-first — API keys in OS keychain, nothing on our servers
  • Free and open source (AGPL-3.0)
